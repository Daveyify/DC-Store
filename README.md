# DC STORE

E-commerce de tecnología y gadgets desarrollado con React + Vite.

## 🔗 Demo
[Ver proyecto en Vercel](URL_DE_TU_PROYECTO)

## 🛠️ Tecnologías
- React
- Vite
- React Router DOM
- JavaScript ES6+

## ✨ Funcionalidades
- Catálogo de 6 productos con imagen, nombre, precio y descripción
- Carrito de compras con contador y total
- Navegación entre páginas con React Router
- Diseño responsive

## 📁 Estructura del proyecto
